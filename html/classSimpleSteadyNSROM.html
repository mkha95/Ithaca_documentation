Detailed Description

Implementation of class for the reduced model of a Steady navier stokes problem for which the online solve uses a consistent approach("segregatedROM")

In this class are implemented the methods for the offline solve and a continuation method for a steady NS problem. this class inherits from reducedSimpleSteadyNS
